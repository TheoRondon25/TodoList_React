# 📝 Todo List - React + Vite + TypeScript

## 🚀 Sobre o projeto

Este projeto é uma aplicação de **lista de tarefas (Todo List)** desenvolvida utilizando **React**, **Vite** e **TypeScript**, como parte da minha formação em React pela **Rocketseat**.

A proposta é construir uma aplicação **completa, escalável e bem estruturada**, aplicando na prática os principais conceitos do ecossistema React moderno.

> ⚠️ Projeto em desenvolvimento

---

## 🎯 Objetivo

O objetivo principal deste projeto é:

* Praticar e consolidar conhecimentos em **React**
* Utilizar **TypeScript** para tipagem segura
* Aplicar boas práticas de **componentização**
* Criar uma aplicação real com **funcionalidades completas de uma Todo List**
* Estruturar o projeto de forma profissional

## ▶️ Como rodar o projeto

### 📦 Pré-requisitos

* Node.js (versão 18+ recomendada)
* pnpm instalado globalmente

---

### 🔧 Instalação

```bash
# Clone o repositório
git clone https://github.com/TheoRondon25/TodoList_React.git

# Acesse a pasta do projeto
cd TodoList_React

# Instale as dependências
npm install

# instale o pnpm
npm i -g pnpm
```

---

### ▶️ Executando o projeto

```bash
pnpm run dev
```
